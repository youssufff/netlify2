## 📌 Loyiha haqida

Bu loyiha **YouTube** platformasining frontend klon versiyasi hisoblanadi. Asosiy maqsad — real YouTube interfeysi ko'rinishini HTML, CSS va JavaScript yordamida qayta yaratish.

Loyiha orqali quyidagi frontend ko'nikmalar amalda qo'llanilgan:
- Semantik HTML tuzilmasi
- CSS Flexbox va Grid bilan zamonaviy layout
- Responsive dizayn (mobil va desktop uchun)
- Sidebar navigatsiya
- Video kartochkalar komponenti

## 🚀 Lokal ishga tushirish

```bash
# 1. Reponi clone qiling
git clone https://github.com/username/youtube-clone.git

# 2. Papkaga kiring
cd youtube-clone

# 3. index.html ni brauzerda oching
open index.html
# yoki shunchaki index.html faylini ikki marta bosing
```
